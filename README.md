## 🎯 Περιγραφή Έργου

Η συλλογή αυτή διασώζει προφορικές μαρτυρίες, θρύλους και βιώματα από τη Γορτυνία, αναδεικνύοντας:

- 🏛️ Την τοπική παράδοση και τη γλώσσα της Αρκαδίας
- ⛪ Τα εκκλησιαστικά και λαϊκά έθιμα
- 📜 Ιστορικά γεγονότα (Ελληνοϊταλικός Πόλεμος, Μικρασιατική Καταστροφή, Επανάσταση 1821)
- 👵 Τις οικογενειακές αφηγήσεις και τη λαϊκή σοφία

Ο ιστότοπος είναι φιλικός σε όλες τις συσκευές (κινητά, tablets, υπολογιστές) και παρουσιάζεται με απλή, καθαρή τυπογραφία που σέβεται την αναγνωστική εμπειρία.

## 📚 Περιεχόμενα Συλλογής

Η συλλογή περιλαμβάνει **15 διηγήματα** και επιπλέον υλικό:

### Διηγήματα (01-15)

1. **Η ψυχή του Θοδωράκη στο Ύψωμα 731** - Μαρτυρία από τον Ελληνοϊταλικό Πόλεμο
2. **Το φως της Παναγίας στα βουνά της Αλβανίας** - Θρησκευτικό θαύμα στον πόλεμο
3. **Μαρτυρίες Πείνας στο Χωριό μου** - Η δύσκολη περίοδος της κατοχής
4. **Η Φωτεινή και το μεγάλο ΟΧΙ κάτω από τη σκιά του ευκαλύπτου** - Γυναικεία αντίσταση στη δικτατορία
5. **Ο Αείμνηστος στη μάχη των Πλαταιών** - Αρχαία ελληνική ιστορία
6. **Η ύστατη ανάσα του Θεσπιέα** - Θερμοπύλες και θυσία
7. **Η Λευτεριά της Τριπολιτσάς** - Επανάσταση του 1821
8. **Το δρομολόι του Αϊ-Γιάννη** - Λαϊκός θρύλος
9. **Ο Χριστός και ο Τεμπέλης** - Λαϊκό παραμύθι
10. **Η Ιστορία Του Κυρίου και του Υφαντή** - Παραβολή
11. **Ο παπάς κι' ο Αγάς** - Ιστορικό ανέκδοτο
12. **Το Θαύμα του Δεκαπενταύγουστου** - Θρησκευτικό έθιμο
13. **Το πρόσφορο της γιαγιάς Φωτεινής και τα θρησκευτικά έθιμα της Γορτυνίας** - Οικογενειακή παράδοση
14. **Το Δωδεκαήμερο στα ορεινά χωριά της Γορτυνίας** - Χριστουγεννιάτικα έθιμα
15. **Το γουρούνι της Αποκριάς** - Παραδοσιακά έθιμα Αποκριάς και σφάξιμο γουρουνιού στη Γορτυνία

### Επιπλέον Υλικό

- **Γλωσσάρι των λέξεων της Γορτυνίας** - Τοπικές λέξεις και ιδιωματισμοί
- **Σχετικά με τον Συγγραφέα** - Βιογραφικό
- **Ευχαριστίες** - Προς τους αναγνώστες

## 🚀 Τοπική Προβολή

Για να προβάλετε το βιβλίο τοπικά:

### 1. Κλωνοποίηση

```bash
git clone https://github.com/Michelis2023/oi-istories-pou-elege-i-giagia.git
cd oi-istories-pou-elege-i-giagia
```

### 2. Άνοιγμα στον Browser

**Επιλογή Α:** Απευθείας άνοιγμα
- Ανοίξτε το `index.html` στον browser σας

**Επιλογή Β:** Με local server (προτείνεται)

```bash
# Python 3
python -m http.server 8000

# ή Python 2
python -m SimpleHTTPServer 8000
```

Στη συνέχεια επισκεφθείτε: `http://localhost:8000/index.html`

## 🧭 Πλοήγηση

- **Αρχική σελίδα** (`index.html`): Πίνακας περιεχομένων με όλα τα διηγήματα
- **Κάθε διήγημα**: Περιλαμβάνει πλοήγηση προς:
  - 📋 Περιεχόμενα
  - ⬅️ Προηγούμενο διήγημα
  - ➡️ Επόμενο διήγημα
- **Επιπλέον υλικό**: Γλωσσάρι, Σχετικά με τον Συγγραφέα, Ευχαριστίες

Ο αναγνώστης μπορεί να διαβάσει:
- Γραμμικά (από το 01 μέχρι τις Ευχαριστίες)
- Επιλεκτικά (ανά θέμα ή ενδιαφέρον)
