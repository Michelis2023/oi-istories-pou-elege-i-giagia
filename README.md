# Οι ιστορίες που έλεγε η γιαγιά

Στατικό διαδικτυακό βιβλίο με λαϊκά και παραδοσιακά διηγήματα εμπνευσμένα από τη Γορτυνία Αρκαδίας, τη θρησκευτική ζωή, την τοπική ιστορία και τις μνήμες των παλαιότερων.

Ζωντανή έκδοση (GitHub Pages):  
https://michelis2023.github.io/oi-istories-pou-elege-i-giagia/

***

## Περιγραφή έργου

Το αποθετήριο αυτό φιλοξενεί μια συλλογή διηγημάτων σε μορφή στατικού ιστότοπου.

Στόχος είναι:

- να διασωθούν προφορικές μαρτυρίες, θρύλοι και βιώματα από τη Γορτυνία,
- να αναδειχθεί η τοπική παράδοση, η γλώσσα και τα εκκλησιαστικά/λαϊκά έθιμα,
- να παρουσιαστεί το υλικό σε απλή, προσβάσιμη μορφή, φιλική σε κινητά, tablet και υπολογιστές. [github](https://github.com/othneildrew/Best-README-Template)

Ο ιστότοπος αποτελείται από απλά HTML αρχεία και ένα ενιαίο φύλλο στυλ (CSS), χωρίς frameworks ή πολύπλοκο build· μπορεί να κλωνοποιηθεί και να λειτουργήσει τοπικά με έναν απλό static server ή απευθείας μέσω GitHub Pages. [docs.github](https://docs.github.com/en/pages/getting-started-with-github-pages/creating-a-github-pages-site)

***

## Δομή περιεχομένου

Κάθε διήγημα βρίσκεται σε δική του σελίδα (01–15), ενώ υπάρχουν επιπλέον σελίδες για Επίλογο, Βιογραφικό, Γλωσσάρι και Ευχαριστίες.

Κύρια αρχεία:

- `index.html` – Αρχική σελίδα, περιγραφή του βιβλίου και πίνακας περιεχομένων.
- `01.html`–`15.html` – Τα επιμέρους διηγήματα.
- `16.html` – Επίλογος.
- `17.html` – Σχετικά με τον συγγραφέα.
- `18.html` – Γλωσσάρι λέξεων της Γορτυνίας.
- `19.html` – Ευχαριστίες προς τον αναγνώστη.
- `assets/css/style.css` – Κεντρικό φύλλο στυλ (τυπογραφία, διάταξη, πλοήγηση).

Όλες οι σελίδες μοιράζονται την ίδια βασική δομή (header, κείμενο, πλοήγηση, footer), ώστε ο αναγνώστης να έχει αίσθηση ενιαίου βιβλίου.

***

## Τεχνολογίες

- **HTML5** – Δομή και σήμανση του περιεχομένου.
- **CSS3** (χωρίς framework) – Τυπογραφία, χρώματα και διάταξη σελίδας.
- **GitHub Pages** – Φιλοξενία του στατικού ιστότοπου. [docs.github](https://docs.github.com/en/pages/getting-started-with-github-pages/creating-a-github-pages-site)

Δεν χρησιμοποιούνται:

- JavaScript,
- βάσεις δεδομένων,
- εξωτερικά CSS frameworks.

Το project είναι εσκεμμένα ελαφρύ, διαφανές και εύκολο στη συντήρηση. [github](https://github.com/othneildrew/Best-README-Template)

## Τοπική προβολή

Για να προβάλετε το βιβλίο τοπικά στον υπολογιστή σας:

1. Κλωνοποιήστε το αποθετήριο:

   ```bash
   git clone https://github.com/Michelis2023/oi-istories-pou-elege-i-giagia.git
   cd oi-istories-pou-elege-i-giagia
   ```

2. Ανοίξτε το `index.html` απευθείας στον browser  
   ή σερβίρετε τον φάκελο με έναν απλό static server (π.χ. Python):

   ```bash
   python -m http.server 8000
   ```

3. Επισκεφθείτε στο πρόγραμμα περιήγησης:  
   `http://localhost:8000/index.html`

## Χρήση και πλοήγηση

- Από την αρχική σελίδα (`index.html`) ο αναγνώστης επιλέγει οποιοδήποτε διήγημα από τον αριθμημένο πίνακα περιεχομένων.
- Σε κάθε σελίδα διηγήματος, στο κάτω μέρος υπάρχουν σύνδεσμοι πλοήγησης:
  - «Περιεχόμενα»
  - «Προηγούμενο διήγημα»
  - «Επόμενο διήγημα»
- Το γλωσσάρι (`18.html`) διευκολύνει στην κατανόηση τοπικών λέξεων και ιδιωματισμών που εμφανίζονται στα κείμενα.

Έτσι ο επισκέπτης μπορεί να διαβάσει το έργο είτε γραμμικά (από το 01 μέχρι τον Επίλογο) είτε επιλεκτικά, ανά διήγημα.

## Δομή φακέλων

Απλή δομή, κατάλληλη για στατικό site:

```text
oi-istories-pou-elege-i-giagia/
├── index.html
├── 01.html
├── 02.html
├── ...
├── 19.html
└── assets/
    └── css/
        └── style.css
```

Δεν υπάρχουν build steps, dependencies ή ειδική ρύθμιση. Το μόνο που απαιτείται είναι ένας HTTP server (ή απλό άνοιγμα των HTML αρχείων σε browser). [tilburgsciencehub](https://www.tilburgsciencehub.com/topics/collaborate-share/share-your-work/content-creation/readme-best-practices/)
